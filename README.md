In this assignment I was tasked to analyse stock market data with given data for the years, 2018, 2019 and 2020. I used Visual Basic Application (VBA) in Microsoft Excel to create the coding for my analysis. It uses For Next loop and conditional fuctions to do the analysis. From the given data, my VBA script mainly calculates and projects the following information once the code is run in the spreadsheet: (1)change in stock price, (2) percent change in stock price, and (3) total volume of stock for each ticker (the company) at the end of each year. The VBA script also further calculates and projects the greatest Increase/Decrease in percent change in price as well the total value of stock for each ticker at the end of the year. In the VBA-Challenge repository, I have uploaded the VBA script file (the .bas file) as well as screenshots of the results I acheived for the 3 different years of data after running my code. In the screenshots, you will see I have used conditional formatting in my VBA script to highlight positive changes in green and negative changes in red.
